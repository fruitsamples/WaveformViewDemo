### WaveformViewDemo ###

===========================================================================
DESCRIPTION:

The WaveformViewDemo project provides a sample audio unit with a custom Cocoa view to draw the incoming audio as a waveform.

===========================================================================
BUILD REQUIREMENTS:

Mac OS X v10.6 or later

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X v10.6 or later

===========================================================================
PACKAGING LIST:

WaveformViewDemo.h
WaveformViewDemo.cpp
- Main code for the audio unit processing and manipulation

CAWaveformView.hCAWaveformView.mm
- View class for processing the waveform display

WaveformViewDemoView.hWaveformViewDemoView.mm
- Class for the Cocoa view

WaveformViewDemoViewFactory.hWaveformViewDemoViewFactory.mm
- Class for the view factory
